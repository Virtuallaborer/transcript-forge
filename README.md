# Transcript Forge

**YouTube → Summary → Action Items → Build Prompts**

A local-first tool that turns any YouTube video into:
- Clean transcripts
- Concise summaries  
- Key action items
- Ready-to-use build prompts for AI coding assistants

Built with **Ollama**, **youtube-transcript-api**, and **Streamlit**.

![Demo](screenshot.png) <!-- screenshot later -->

## Features

- Works with **any YouTube video** (short or long)
- Automatic **chunking** for long videos
- **Download all** as ZIP or individual Markdown files
- Local & private (no data leaves your machine)
- Customizable model & tone
